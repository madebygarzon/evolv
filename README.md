# Evolv — AI-Powered Personal Growth Platform

Full-stack product focused on helping users grow through structured goals, coaching prompts, and progress tracking.

## Why this project is recruiter-relevant
- Product-minded implementation beyond a demo template.
- Combines app architecture, UX flow, and data persistence.
- Strong baseline for scaling into a SaaS platform.

## Tech Stack
- **Frontend:** Next.js (App Router), React, TypeScript
- **Backend/Data:** Prisma
- **UI:** Component-based architecture

## What it does
- Structured personal-growth journeys
- AI-assisted coaching flows
- Goal/progress tracking experience
- Reusable product components and routes

## Project Structure
- `app/` → routes and page composition
- `components/` → reusable UI elements
- `lib/` → utilities/shared logic
- `prisma/` → schema and data layer

## Run Locally
```bash
npm install
npm run dev
```
Then open `http://localhost:3000`.

## Why it matters
Evolv demonstrates the ability to move from idea to product foundation, balancing delivery speed with maintainable architecture.
